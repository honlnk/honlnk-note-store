# Umo Editor 方法说明

## 示例代码

```vue
<template>
  <umo-editor
    ref="editorRef"
    v-bind="options"
  />
  <button @click="getOptions">获取配置</button>
</template>

<script setup>
import { ref } from 'vue';
import { UmoEditor } from '@umoteam/editor';

const editorRef = ref(null);
const options = ref({
  // 配置项
  // ...
});

const getOptions = () => {
  const options = editorRef.value.getOptions();
  console.log(options);
};
</script>
```

---

## 方法说明

---

### `getOptions`
**说明**：获取 Umo Editor 的配置。

**参数**：无

**返回值**：包含 Umo Editor 配置的 `Object`。

---

### `setOptions`
**说明**：设置 Umo Editor 的配置。

**参数**：
- `options`：包含 Umo Editor 配置的 `Object`。

**返回值**：包含 Umo Editor 配置的 `Object`。

---

### `setToolbar`
**说明**：设置 Umo Editor 的工具栏类型。

**参数**：
- `mode`：String，工具栏类型，可选值为 `ribbon`、`classic`。
- `show`：Boolean，是否显示工具栏。

**返回值**：无。

---

### `setPage`
**说明**：设置 Umo Editor 的页面信息。

**参数**：
- `size`：String，页面大小（如 A4）。
- `orientation`：String，页面方向（`portrait`/`landscape`）。
- `background`：String，CSS 颜色值。

**返回值**：无。

---

### `setWatermark`
**说明**：设置 Umo Editor 的水印信息。

**参数**：
- `type`：String，水印类型（`compact`/`spacious`）。
- `alpha`：Number，透明度（0-1）。
- `fontColor`：String，CSS 颜色值。
- `fontSize`：Number，字体大小（px）。
- `fontFamily`：String，CSS 字体。
- `fontWeight`：String，字体粗细（`normal`/`bold`/`bolder`）。
- `text`：String，水印内容（建议不超过 30 字符）。

**返回值**：无。

---

### `setDocument`
**说明**：设置 Umo Editor 的文档信息。

**参数**：
- `title`：String，文档标题。
- `bubbleMenu`：Boolean，是否启用气泡菜单。
- `blockMenu`：Boolean，是否启用块级菜单。
- `markdown`：Boolean，是否启用 Markdown 语法。
- `spellcheck`：Boolean，是否启用拼写检查。

**返回值**：无。

---

### `setContent` #插入内容
**说明**：设置 Umo Editor 的内容。

**参数**：
- `content`：HTML 或 Tiptap 文档格式。
- `options`：
  - `emitUpdate`：Boolean，是否触发更新事件。
  - `focusPosition`：String/Number，聚焦位置（`start`/`end`/`all`）。
  - `focusOptions`：Object，聚焦配置（如 `{ scrollIntoView: true }`）。

**返回值**：无。

---

### `insertContent` #插入内容（选中位置）
**说明**：在当前选区插入内容（v6.0.0 新增）。

**参数**：
- `content`：HTML 或 Tiptap 文档格式。
- `options`：
  - `updateSelection`：Boolean，是否更新选区。
  - `focusPosition`：String/Number，聚焦位置。
  - `focusOptions`：Object，聚焦配置。

**返回值**：无。

---

### `setLocale`
**说明**：设置 Umo Editor 的显示语言（v2.0.0 新增）。

**参数**：`zh-CN` 或 `en-US`。

**返回值**：无。

---

### `setTheme`
**说明**：设置 Umo Editor 的主题（v2.1.0 新增）。

**参数**：`light`、`dark`、`auto`。

**返回值**：无。

---

### `getPage`
**说明**：获取 Umo Editor 的页面配置信息（v5.0.0 新增）。

**参数**：无。

**返回值**：页面配置信息。

---

### `getContent` #获取内容（html/json/text）
**说明**：获取 Umo Editor 的内容。

**参数**：String，格式类型（`html`/`json`/`text`）。

**返回值**：对应格式的内容。

---

### `getContentExcerpt` #获取内容摘要
**说明**：获取内容摘要。

**参数**：
- `charLimit`：Number，默认 100 字符。
- `more`：String，结尾标识（默认 `...`）。

**返回值**：文档摘要。

---

### `getHTML` #获取内容（html） 
**说明**：获取 HTML 内容。

**参数**：无。

**返回值**：HTML 字符串。

---

### `getJSON` #获取内容（json） 
**说明**：获取 JSON 内容。

**参数**：无。

**返回值**：JSON 数据。

---

### `getText` #获取内容（text）
**说明**：获取纯文本内容。

**参数**：无。

**返回值**：文本字符串。

---

### `getImage` #获取页面截图
**说明**：获取页面截图。

**参数**：String，格式（`blob`/`jpeg`/`png`）。

**返回值**：图片信息。

---

### `saveContent` #保存
**说明**：保存内容。

**参数**：Boolean，是否显示成功提示。

**返回值**：无。

---

### `getEditor`

**参数**：无。

**返回值**：Tiptap Editor 的 Vue 实例。

---
